Detailed Description

Represents a Principal in the Kerberos database.

This class should be your main interface to modify the Kerberos database. Together with the Connection class, all operations available in the kadmin commandline program may also be performed using these classes. Single exception is the (local) database dump.

To reduce network load, data will be fetched from the server only if needed.

Any changes will be invisible to the server until commit_modifications() is called.

Passwords will be wiped from memory after successful transmission to the server.

Small usage example:

boost::smart_ptr<Connection> pc( Connection::from_password("adminpw") );
boost::smart_ptr<Principal> pp( pc->get_principal("foo") );
pp->set_name("bar"); // rename foo -> bar
pp->set_password("secret"); // change password
pp->commit_modifications(); // make changes visible
