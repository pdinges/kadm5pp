1 kadm5pp
2 =======
3 
4 The kadm5pp library provides an object-oriented interface to the
5 [Heimdal][heimdal]
6 [Kerberos&nbsp;5 administration libraries][kadm5]. It is written in
7 in C++ using [Boost][boost] and has an STL-style API. It simplifies
8 the overall handling by automating context management and hiding the
9 exhausting C-style error handling behind exceptions. It also tries to
10 motivate the right order of object use through its interface.
11 
12 All objects are also exported to a [Python&nbsp;2][python] module
13 through Boost.Python. Integration into Python makes scripting
14 administrative tasks more comfortable than calling the `kadmin`
15 command from shell scripts.
16 
17 The library currently supports only a limited (but useful) set of
18 operations on the Kerberos database. I have no intention of
19 developing the library any further and publish it here in the hope
20 that it might be useful to someone.

23 Installation
24 ------------
25 
26 Building the module requires the following:
27 
28 * A C++ compiler and linker, as well as Make (`build-essential`)
29 * The [Heimdal][heimdal] Kerberos&nbsp;5 libraries and development
30  headers (`heimdal-dev`).
31 
32  **Note:** The MIT Kerberos distribution will *not* work because the
33  headers have different names and lack some of the used data
34  structures.
35 * The [Boost][boost] library and associated headers (`libboost-dev`)
36 
37 The name in parentheses after each item is the name of the
38 Debian/Ubuntu Linux package that contains the required files. The
39 library has been verified to build against Heimdal&nbsp;1.5.2 and
40 Boost&nbsp;1.5.0.
41 
42 Instead of installing Heimdal and Boost as system-wide libraries, you
43 can also compile and install them in a local directory. Adjust the
44 `include_dirs` (`-I...`) and `lib_dirs` (`-L...`) variables in the
45 `Makefile` to tell the compiler about their locations.
46 
47 After all dependencies have been installed, the kadm5pp library can
48 be built using Make. Simply execute `make` in the `src/` directory.

51 Usage Example
52 -------------
53 The following Python code renames principal `foo` to `bar`:
54 
55 ~~~~{.py}
56 from kadm5 import Connection, Principal
57 
58 c = Connection.from_password("adminpw")
59 p = c.get_principal("foo")
60 p.set_name("bar")
61 p.commit_modifications()
62 ~~~~
63 
64 See the [API documentation][kadm5pp-api] of the `Principal` class for
65 the same example in C++.
