Constructs a new CCacheContext with the given connection data.

Parameters
ccnameUse credentials from this cache. The name may be a filename or any other valid identifier of form TYPE:ID that the libraries understand. If TYPE: is omitted, FILE: is assumed.
realmThe Kerberos realm for this context. It will be used as default for all principals without specified realm. If empty, the default realm will be used.
hostHostname of the KAdmin server to which to connect. If empty, defaults to the used realm's admin_server config parameter.
portThe KAdmin server's port number. If 0, the libraries' default port number is used.
