Parameters
passwordConnect to the KAdmin server using this password.
clientThe name of the Kerberos principal to authenticate as. If missing, the libraries' default value will be used.
realmThe realm to assume if this part of a Principal's name is omitted. If missing, the libraries' default value will be used.
hostHostname of the KAdmin server to connect to. If missing, defaults to the admin_server parameter in the Kerberos configuration.
portThe KAdmin server's port number. If 0, uses the libraries' default port number.
