Detailed Description

Base class for all exceptions related to Kerberos and KAdmin functions.

It contains an error code member that holds the error number returned by the Kerberos and KAdmin functions. See <krb5_err.h> and <kadm5/kadm5_err.h> for relevant #defines. Use error_code() to retrieve the error number.

Note
Some exceptions may return error code 0 if their class does not correspond to a library error. (An example for this is AlreadyExists).

Also, a convenience function throw_on_error() is provided to wrap library function calls. It will throw an appropriate exception if an error code was returned. See throw_on_error()'s documentation for an example.
